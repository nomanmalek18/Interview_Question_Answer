1. What is HTML

HTML stands for Hyper Text Markup Language. HTML is the standard markup language for creating Web pages. HTML is used to structure a web page and its content. For example, content could be structured within a set of paragraphs, a list of bulleted points, or using images and data tables.

21. What is the use of Anchor tag? Explain two ways to specify URL in href attribute?

The anchor tag in HTML is used to create links to a target URL, such as a web page, file, or email address. The href attribute of the anchor tag specifies the URL of the target.

Absolute URL:-

Points to a file on another website. For example, href="http://www.example.com/theme.css".

Relative URL:-

Points to a file within the same website. For example, href="/themes/theme.css". Relative URLs are shorthand that refer to a file in relation to the current file's location.
