2. how to fetch html elements

You can use the getElementsByClassName() method to access elements by their class name. You can also use the querySelector() method to select an element based on its CSS selector. The most common way to access an element is by using its ID, with the getElementById() method.

3. what is a variables

A JavaScript variable is simply a name of storage location. There are two types of variables in JavaScript : local variable and global variable. There are some rules while declaring a JavaScript variable (also known as identifiers). Name must start with a letter (a to z or A to Z), underscore( _ ), or dollar( $ ) sign.
