5. array and object

Arrays and objects are both used to store data, but they differ in how they store it and how they are used:

Arrays:-

Used to store a list of elements with the same data type or structure. Arrays store data in consecutive memory locations and use numbered indexes. For example, an array of numbers can be declared using square brackets.

Objects:-

Used to group multiple sets of data that belong together. Objects store data in named key-value pairs and use named indexes. For example, an object can be declared using curly brackets.

differences between arrays and objects:-

Properties: Objects tend to use names for their properties, while arrays use numbers starting from 0

Values: Arrays usually contain varying amounts of conceptually identical values.

Methods: Methods are functions that live in properties and act on the value of which they are a property.

Iteration: You can iterate over arrays using a special kind of for loop.

6. what is hoisting in javascript

Hoisting is a JavaScript mechanism where variables and function declarations are moved to the top of their scope before code execution. This means that you can use a variable or function before it's declared in your code.

Important Points:

Only Declarations are Hoisted: While the declaration is hoisted, the assignment (initialization) is not.

Function Declarations vs. Function Expressions: Function declarations are hoisted completely, allowing you to call them before they are defined. Function expressions (assigned to a variable) are not hoisted in the same way.

var vs. let and const: Variables declared with var are hoisted to the top of their scope and initialized with undefined. Variables declared with let and const are also hoisted, but they are not initialized. Accessing them before declaration results in a ReferenceError

15. What are the following looping structures available in javascripts?

for loop: This is the most common loop, used when you know the exact number of iterations in advance.

while loop: This loop executes as long as a specified condition is true.

do...while loop: Similar to the while loop, but it executes the code block at least once before checking the condition.

for...in loop: This loop iterates over the enumerable properties of an object.

for...of loop: This loop iterates over the values of iterable objects (arrays, strings, maps, sets, etc.).
